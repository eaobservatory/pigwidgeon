Info sections

You can configure multiple Info sections -- each of these allows you to add extra information about a paper. They can be setup to allow either a predetermined set of choices, via radio boxes or check boxes. Alternately they can allow text entry, either through a free form entry box, or a region that expects short texts, entered one per new line.
